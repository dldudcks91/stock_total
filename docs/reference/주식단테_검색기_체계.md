# 주식단테 검색기 · 매매 체계 구조

> 인터넷 공개 자료 기반 정리 (2026-05-12 조사).
> 구체 수식 일부는 유료 강의 영역이라 공개 소스에서 확인되는 범위까지만 정리.

흩어진 단편들을 종합하면 주식단테의 검색기는 **3층 구조**.
**(1) 차트 설정** → **(2) 차트 패턴 용어(검색기 원재료)** → **(3) 기법(=검색기 컨셉)** → **(4) 판매되는 검색기 상품**.

---

## 1. 차트 설정 — 단테 고유 이평 세팅

일반적인 5/10/20/60/120/240 대신 **자기만의 장기 이평 체계**.

| 항목 | 값 |
|---|---|
| 이동평균선 기간 | **5 / 15 / 33 / 56 / 112 / 224 / 448** |
| 종류 | 지수이동평균(EMA) |
| 가격 | 종가 기준 |
| 강조선 | **112(녹색) / 224(검정) / 448(빨강), 굵게** |
| 보조지표 | 일목균형표(기준선 26, 후행스팬 26), 자체 "파란점선" |

> "파란점선"은 단테가 만든 자체 보조지표로, 일목균형표/장기이평을 가공해 **매수자리**를 점선으로 표시. 외부 사용자가 [앙지표 형태로 역구현해 공유](https://damoang.net/free/5765882)하기도 함.

---

## 2. 차트 패턴 용어 (검색기를 짜는 원재료)

| 용어 | 정의 |
|---|---|
| **역배열** | 큰 이평선(448→224→112) 위, 작은 이평선(5/15) 아래. 저평가·하락 추세 신호 |
| **매집봉** | 세력 매집 캔들. 윗꼬리 단 장대봉 + 볼린저밴드 상단 돌파, 거래량 급증 |
| **공구리** | 직전 박스권(언덕)을 뚫고 올라온 자리, 224일선 아래 박스 돌파. "바닥을 잡는 게 아니라 만들고 올라오는 자리" |
| **파란점선** | 자체 보조지표로 그려지는 매수 자리 표시 |
| **112** | 112일선(녹색). 단테 체계의 기준선 |
| **밥그릇** | 4구간 누적 패턴 (1.가격조정 → 2.기간조정 → 3.흔들기/매수 → 4.급등) |
| **짝궁뎅이 / 쌍바닥 / 역헤드앤숄더** | 3번 구간 확인용 보조 패턴 |
| **하이힐** | 낙폭과대 후 공구리 지지 받아 형성되는 형태 |

**"역매공파"** = **역**배열 + **매**집봉 + **공**구리 + **파**란점선 4종 동시 충족 자리. 단테 매수 타점의 대표 시그니처.

---

## 3. 기법 = 검색기 컨셉

| 기법 | 적합 주기 | 진입 조건 | 청산 |
|---|---|---|---|
| **256 기법** | 단·중·장 | 밥그릇 2번 끝단 + 역배열에서 **5일선이 20일선 골든크로스** | 60일선 돌파에서 익절 (중장기는 112/224일선으로 변형) |
| **이평때리기** | 단기 | 회복 국면, 역배열 상태에서 **112일선 돌파** | 224일선 매도 → 지지 시 재매수 → 448일선 매도 |
| **데드 기법** | 스윙 | 50%+ 급등 후 2파동 이상, **부채꼴 모양 닫힐 때** 112/224/448 터치 | "닿으면 90% 반등" (단, 엘리어트 3파·쌍봉·헤숄·다중 터치 제외) |
| **공구리 / 하이힐 기법** | 단·스윙 | 직전 언덕(박스) 돌파 + 224일선 아래 박스 탈출 | 매물대 저항 |
| **밥그릇 기법** | 6개월+ 중장기 | 3번 구간(흔들기) 진입, 224일선 돌파 시 공략 | 4번 급등 구간에서 50~100% 익절 |
| **종가베팅** | 익일 단타 | 장 마감 직전 종가에 세력 수급이 들어오는 자리 | 시간외 또는 다음날 시가 갭 |

## 5. 한 줄 요약

**"역매공파(역배열·매집봉·공구리·파란점선)"** 4종 시그널을 기본 골격으로,
**112/224/448 EMA**라는 비표준 장기 이평선 위에서
**밥그릇(중장기) / 256·이평때리기 / 데드(스윙) / 종가베팅(단타)** 4계열의 검색기를 구성하는 체계.
